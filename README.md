### 2. Giao diện (`main.dart`)

#### Widget chính:

* `MyApp`: cấu hình app
* `MyHomePage`: màn hình chính
* `Tile`: hiển thị từng chữ

#### Tile UI:

* Màu sắc:

  * Xanh lá: đúng vị trí
  * Vàng: đúng chữ sai vị trí
  * Xám: sai
  * Trắng: chưa nhập
